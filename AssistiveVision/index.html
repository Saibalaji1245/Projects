Welcome to Assistive Vision

Your audio-first tool for navigating the world. This application uses AI to identify objects and provide audio alerts.

How It Works

  1. Navigate: Use the navigation buttons or voice commands (Press 'V') to move between pages. Say "Help" at any time to hear your options.
  2. Detect Objects: Go to the "Upload" page to submit a pre-recorded video clip and hear audio alerts for specific objects.

Get Started

You can explore the "About" page to learn more about the technology. If you are a new user, please "Register". If you are returning, "Login" to access your profile and upload history.

Press 'R' at any time to hear a list of all interactive elements on your current page.

Press 'T' or use the button in the header to toggle between classic and dark themes.

About the Project

This tool is designed to empower visually impaired users by providing an extra layer of environmental awareness.

Core Technology

The application uses a computer vision model (simulated in the 'Upload' section) to identify key objects for navigation, such as vehicles, pedestrians, and crosswalks.

An offline text-to-speech (TTS) engine prioritizes and announces all AI-generated information. The entire interface is designed to be fully operable via keyboard and voice commands.

Future Goals

We are actively working on expanding this platform. Our next major goal is to implement real-time object detection directly from a device's camera, providing instant feedback while a user is walking. We also plan to improve object tracking and add more object categories, such as 'Stairs' or 'Obstacle'.
